A Sample Map

Welcome to this demo of MapFrame!

MapFrame is a lightweight CSS stylesheet designed to help you quickly create a web map.

It's designed to be used as a wireframe, and then forked to create custom versions, or replaced entirely beyond the Proof-of-Concept stage.

Attribution

There's no need to credit MapFrame in your map creations.

I'd be greatful if you could drop a reference back to the repository, if you found it useful.
